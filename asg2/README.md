# Assignment 2 - Blocky 3D Animal

3D animated elephant with hierarchical modeling.

## Features

- 3D elephant with 12 body parts
- Hierarchical joint control with sliders
- Walking animation with natural movement
- Mouse drag to rotate view
- Shift+click poke animation
- Cylinder primitives for legs
- Performance optimized (60+ FPS)

## Files

- `asg2.html` - Main page
- `asg2.js` - WebGL 3D logic
