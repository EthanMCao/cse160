# Assignment 1 - WebGL Paint & Shapes

WebGL paint program with interactive shape drawing.

## Features

- Draw points, triangles, and circles
- RGB color sliders
- Size and segment controls
- Click and drag to paint
- Custom picture with initials "EC"
- Smooth stroke interpolation

## How to Run

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080/asg1.html`

## Files

- `asg1.html` - Main page
- `asg1.js` - WebGL logic
- `Point.js`, `Triangle.js`, `Circle.js` - Shape classes
- `img.jpg` - Hand-drawn reference
